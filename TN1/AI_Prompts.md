# Prompts IA utilisés — TN1

Exemples des prompts utilisés avec l'assistant IA (GitHub Copilot) lors de la réalisation du TN1.

**Note importante :** Chaque résultat généré par l'IA a été systématiquement révisé, testé et validé avant d'être intégré au projet. L'IA a été utilisée comme outil d'assistance, et non comme source de vérité. Toutes les suggestions ont été évaluées avec un esprit critique pour garantir leur exactitude, leur pertinence et leur conformité aux exigences du travail.

## Structure et setup

- « Comment structurer un projet Go avec des tests unitaires ? »
- « Comment configurer GitHub Actions pour exécuter mes tests automatiquement ? »
- « Comment intégrer Codecov avec mon workflow GitHub Actions ? »

## Tests unitaires

- « Est-ce que mes tests couvrent suffisamment de cas pour la fonction DaysUntilDeadline ? »
- « Quels cas limites devrais-je ajouter pour améliorer ma couverture ? »
- « Peux-tu me créer un fichier table-driven pour que je puisse comparer la structure avec mon fichier deadline_test.go ? »
- « Quelles années sont bissextiles pour que je puisse les utiliser dans mes tests ? »
- « Est-ce que je gère correctement les années bissextiles dans mes tests ? »

## Couverture et qualité

- « Comment créer un badge de couverture dans GitHub ? »
- « Comment atteindre 100% de couverture de code en Go ? »
- « Comment générer un rapport de couverture avec go test ? »

## Debugging et format

- « Pourquoi mon test échoue avec le format de date 2025/05/26 ? »
- « Comment valider strictement le format YYYY-MM-DD en Go ? »

## Documentation et vidéo

- « Peux-tu m'écrire un script pour présenter mes tests unitaires ? »
- « Peux-tu numéroter mes tests dans le code ? »
